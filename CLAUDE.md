# Convallaria for Claude Code

Before working in this repository, read `AGENTS.md` at the repository root and follow it as the project authority.

Use the commands in `claude/commands/` as lightweight Claude Code entry points. They route to the shared skill at `skills/convallaria/SKILL.md`.

Keep `claude/` as the tracked adapter source. `.claude/` is generated by `python3 scripts/conva.py sync --no-codex-cachebuster` and may contain local settings such as `.claude/settings.local.json`; do not treat `.claude/` as source.

Do not edit `skills/convallaria/subskills/design-refine/` unless the user explicitly asks to update that vendored subskill.
