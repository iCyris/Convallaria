# Atelier for Claude Code

Atelier is an agentic design suite. Use the commands in `claude/commands/` as lightweight entry points, then read the shared skill at `skills/atelier/SKILL.md`.

Core rule: keep generated project content in the user's requested language, but keep Atelier source files in English.

The vendored `skills/atelier/subskills/design-refine/` folder is copied from the original design-refine skill and should not be modified unless the user explicitly asks to update that subskill.
